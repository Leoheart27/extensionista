Descubra oque é uma Linguagem de Programação

Bom, vamos lá. Provavelmente voce já deve ter visto alguém que trabalha com tecnologia olhando para uma tela cheia de códigos que não parecem fazer sentido algum? Você consegue se imaginar escrevendo um código dessa maneira? Não? Pois bem, aqui vamos te ajudar a dar os primeiros passos.

Conceito de Linguagem de Programação

É uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software). Um software pode ser desenvolvido para rodar em um computador, dispositivo móvel ou em qualquer equipamento que permita sua execução. Elas foram criadas para solucionar qualquer tipo de problema na área tecnológica computacional e estão em constante evolução, sempre buscando oferecer as mais completas ferramentas para as mais complexas tarefas. É fato que não existe nenhuma linguagem ideal para todos os cenários de desenvolvimento. Cada uma contém suas peculiaridades e características que as tornam boas e populares em certos ambientes, segue abaixo as mais conhecidas e utilizadas.

Tipos de Linguagens

Possivelmente você já ouviu falar de várias. No começo deste post existe uma imagem com algumas delas. Javascript,C++, Python e PHP esses são exemplos de Linguagens de Programação. Vale a pena ressaltar que existem linguagens no mundo TI que não sao Linguagens da Progamação, como HTML e CSS. elas são usadas para estilização e estruturação de uma página web. Ainda que existam inúmeras linguagens, segue a lista das principais:

  • Javascript
  • Python
  • PHP
  • C++
  • Java

Javascript

Pode ser facilmente confundida com Java, mas lembre-se: essas linguagens são totalmente diferentes uma da outra. É uma linguagem desenvolvida para navegadores e é a Programação Web mais popular do mercado. JavaScript permite até manipular o comportamento do navegador. É possível criar janelas pop-up, exibir mensagens, alterar dimensões etc., além de interagir com formulários, realizar cálculos e fazer validações.
