    Variáveis

    Variáveis são formas de salvar dados em nosso programa.
    Uma variável tem um nome que nós escolhemos para indicar o tipo de data que está nela. Poderiamos criar uma variável de nome minha_idade e atribuir o valor de 20 ou uma variável de nome meu_nome e atribuir "Lucas".
    Para criar uma variavel devemos seguir a seguinte sintaxe:

    my_number = 10
    my_float_number = 1.0123
    my_text = "Texto"

    Podemos usar o comando print para visualizarmos nossa variavel no console.

    print(my_number) # 10
    print(my_float_number) # 1.023
    print(my_text) # Texto

    Existem diferentes tipos de dados em python:

  • String = "Texto"
  • Int = 10 número inteiro
  • float = 1.5 número decimal

    Operadores Matemáticos

    Vamos praticar o uso de variaveis com um exercicio de matemática.
    Se você não gosta de matemática, não se assuste! você não precisa ser um expert em matemática para saber programar.
    Vamos começar com um exemplo de calculo simples.

    Exemplo:

    x = 50
    y = 10

    print(x - y) # 40
    print(x + y) # 60
    print(x / y) # 5
    print(x * y) # 500

    OBSERVAÇÃO! Em python expressões númericas seguem a ordem:
    1º Potenciação e Radiciação, 2º Multiplicação e Divisão, 3º Soma e Subtração.

    print((6 + 10 * 2) / 2) # 13

    Operadores + Strings

    podemos também utilizar opreadores matemáticos para "juntar" variaveis do tipo string:

    primeiro_nome = "Lucas"
    segundo_nome = "Ferreira"

    print(primeiro_nome + segundo_nome) # LucasFerreira

    Note que ao "juntar" com operador de adição, ele literalmente uniu as 2 strings, sem espaço entre as duas.
    Uma das formas de se resolver isso, é somando mais uma string com espaço entre as 2.
    Exemplo:

    print(primeiro_nome + " " + segundo_nome)
    # Lucas Ferreira


    F-strings

    Podemos criar strings dinâmicas com variáveis usando f-string. É uma ótima sintaxe para ter conhecimento sobre.

    Assim como vimos anteriormente, podems usar o print para visualizar um valor de uma variável, no caso uma string:

    my_name = "Lucas"

    print(my_name) # Lucas

    Mas podemos usar uma F-string para operarmos melhor essa informação, a sintaxe ficaria:

    my_name = "Lucas"

    print(f"Eu me chamo {my_name}") # Eu me chamo Lucas

    F-string na prática

    Utilizando a sintaxe correta, use o comando print para visualizar 3 variáveis diferentes com f-string.

    nome = "Lucas" # string
    idade = 20 # int
    altura = 1.75 # float

    print("Meu nome é {nome}, tenho {idade} anos e {altura} metros de altura")
    # Meu nome é Lucas, tenho 20 anos e 1.75 metros de altura


    Listas

    Estrutura composta é uma variavel que pode conter diversos valores.

    Lista é uma forma de organizar dados, em outras linguagens podem ter outros nomes como "array", mas em python nós temos lists.
    Muitos aplicativos têm seus itens organizados em listas.
    Por exemplo:

  • Uma lista de produtos em um site de compras
  • Uma lista de postagens nas redes sociais
  • Um jogo de tabuleiro pode ter uma lista de comandos

    • Para declarar uma variavel como uma lista, utilizamos [colchetes], com virgulas separando cada item.

    frutas = ["Maçã", "Laranja", "Banana", "Uva"]

    Uma lista pode conter qualquer tipo de dado, a lista frutas acima, é uma lista de strings

    Acessando uma lista

    Podemos acessar um item de uma lista através de seu indice.
    O primeiro item de uma lista se refere ao indice 0.

    print(frutas[0]) # Maçã
    print(frutas[1]) # Laranja

    Alterando uma lista

    Podemos alterar um item de uma lista de diversas formas, uma delas é pelo seu indice.

    lista original frutas = ["Maçã", "Laranja", "Banana", "Uva"]

    frutas[0] = "Banana"
    print(frutas)
    # ["Banana", "Laranja", "Banana", "Uva"]

    Existem diversas funções e formas de se manipular uma lista, mas não vamos nos aprofundar em todas elas, para não ficar muito massante.

    Loops

    Em Python, existem dois tipos principais de loops: o loop for e o loop while.

    Loop "for"

    O loop for é usado para iterar sobre uma sequência (como uma lista, tupla, string) ou outros objetos iteráveis.
    Vamos usar como exemplo de como uma lista interage com loop for.

    frutas = ["Maçã", "Laranja", "Banana", "Uva"]
    estrutura for 
    for i in frutas:
    print(i)

    # Maçã
    # Laranja
    # Banana
    # Uva

    Após o for nós escolhemos o nome que vai indicar cada item da lista. Nesse caso escolhemos i (podendo ser outra a sua escolha) que representa cada fruta na lista. Vemos que com a função print cada item da lista será chamado, um de cada vez.

    Loop "while"

    O loop while executa um bloco de código repetidamente enquanto uma condição específica for verdadeira.

    Nesse caso iremos criar uma condição utilizando o operador de comparação <

    contador = 0
while contador < 5:
   contador = contador + 1

print(contador) # 5

    Basicamente nós criamos um loop que vai somar 1 à variável contador, enquanto contador for menor que 5.

    Funções

    Em Python, uma função é um bloco de código reutilizável que executa uma tarefa específica. Funções ajudam a organizar o código, tornando-o mais modular, legível e fácil de manter.

    Definindo uma função

    Você pode definir uma função usando a palavra-chave def, seguida pelo nome da função, parâmetros entre parênteses e um bloco de código indentado que compõe o corpo da função. 

      def somar(x, y):
      return x + y
    resultado = somar(10, 90)

    print(resultado) # 100

    Vamos utilizar oque vimos anteriormente e juntar com funções.

    Se eu quero descobrir a área de um circulo eu posso utilizar:

    raio = 5
    area = 3.14 * raio * raio
    print(area) # 78.5

    Agora vamos transformar isso em uma função que pode ser reutilizada com parametros diferentes.

    def area(x):
      pi = 3.14
      area = pi * x * x
      return area
    print(area(5)) # 78.5
    print(area(2)) # 12.56
    print(area(8)) # 200.96

    E se eu tiver uma lista como parametro?

    numeros = [1, 5, 10] 
      def somar(lista):
        x = 10
        for i in lista:
            x = x + i
        return x
    print(somar(numeros)) # 26

    Aqui nós criamos uma função que utiliza um loop for para somarmos a nossa variavel x cada número dentro dela, e retornamos a variavel x modificada.
